Release 13

  • The big one! Added a Timed Review system! (aka Spaced Repetition System, SRS) Correctly drawn characters will now be entered into a time delayed repetition queue. You will be asked to repeat the character at timed intervals over days and months, to maximize your memorization.
  • The next scheduled timed review is now shown prior to 'readings' and 'meanings' after correctly drawing a character.
  • The character set progress activity is now updated with a new colour for 'timed repetition'.
  • In the per character set 'Your Progress' Activity, you can now manually alter a character's progress. Set it to any of the progress states: passed, failed, reviewing, or timed reviewing. Useful if you've forgotten certain characters and want to review more.
